List< Contatto > it.unisa.diem.progettoinf.gruppo25.model.GestoreFile.leggiCSV ( String filename) throws IOException
inline

Legge un file CSV e restituisce una lista di contatti.

Questo metodo legge il contenuto di un file CSV specificato dal parametro filename e lo converte in una lista di oggetti della classe Contatto.

Precondition
Il file specificato da filename deve esistere e deve essere di tipo CVS.
Postcondition
Se l'operazione ha successo, viene restituita una lista di contatti non nulla.
Parameters
[in]filenameNome del file CSV da leggere.
Returns
Lista di contatti letta dal file CSV.
